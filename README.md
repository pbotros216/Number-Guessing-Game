# Number Guessing Game (Binary Search)

This Python program uses a binary search algorithm to guess a number chosen by the user between 1 and 100 in 7 attempts or fewer.

## How it works
- The program calculates the midpoint of a range
- Based on user feedback, it adjusts the range
- Each step cuts the possibilities in half (binary search)

## Key Concepts
- Binary search algorithm
- Control flow (loops and conditionals)
- User input validation

## Why this matters
This project demonstrates problem-solving and algorithmic thinking by applying an efficient search method rather than guessing randomly.
